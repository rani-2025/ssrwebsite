app level secuity

1. user assigned to a profile 2. app assigned to a profile 3. only the users whose profile matches to the app profile can able to view the app is called app level security.

object level secuity

1. created an object with one tab 2. tab assigned to the app 3. even though user has access to the app he cant access the tab until the tab assigned with same profile as user

filed level secuity

1. created an object with salary filed 2. tab created with object and assigned to the app 3. even though user has access to the app he cant access the field until the filed assigned with same profile as user. 4. even though profile matches if thats not added in page layout also user cannt access.
