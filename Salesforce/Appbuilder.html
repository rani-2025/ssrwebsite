Click dropdown besides and choose msi file to down load java 10. Go to downloads folder, do double click on Zulu msi to install java 11. Then in downloads folder do extract all for dataloader_v64.0.1 then go inside click on install to complete the setup

while loading data using dataloader following is the format we need to follow in csv file

1. when created a filed and click on hyperlink u can id in url which is generated by salesforce u can this id as key when u do update, delete, upsert operations from csv file. 2. in salesforce - the one defined in statement 1 considered as id , what ever unique field in object going to be defined as e.g mobileno then that that should be listed under mobileno -External ID 3. for insert / export no ID required but other operations we need to ID. 4. the ID will get created only when insert operation happens. 5. let say for example in csv u define ID for few records and no ID assigned for few, then it will treat what ever record had ID it will consider as update remaining it will treat as insert. 6. let say id we delete any records -> go to recycle in dataloader -> and do un delete -> it will restore the data
