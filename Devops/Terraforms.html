how to create terraform

1. its a tool 2. install it some where (https://developer.hashicorp.com/terraform/tutorials/aws-get-started/install-cli) 3. Harshicorp is the company who created terraform 4. aws console - for programatic access we need to following setup a. create IAM user -> IAM - users -> create terraforn_user , unselect console access then save b. add permission in cole -> attach plicy directory-> select administrator access -> add permission c. create access key -> add some tag value -> create access key for cli , Description tag value(terraforn_tag) accesskey - secretaccesscode - 5. once done go vs code - install aws cli (https://docs.aws.amazon.com/cli/v1/userguide/install-macos.html) 6. go to vs code - aws configure enter aws access key id, secret accesskey, region name - us-east-1, outputformat - json 7. aws ec2 stop-instances \ 'instance id' (command from vscode to stop ec2) 8. how does terraform know u writing script for aws / azure -> u need to install terraform provider (depends on aws/azure/gcp u need to install related providers) (https://registry.terraform.io/browse/providers) in vscode folder -> create provider.tf (name not important extension should be .tf --> inside content is important p-> add ) terraform { required_providers { aws = { source = "hashicorp/aws" version = "6.4.0" } } } provider "aws" { region = "us-east-1" access_key = "" // not a good practice but needed we can } once done -- in terminal run a. terraform init // initialize the code by scan all .tf and try to find required providers and download on laptop
