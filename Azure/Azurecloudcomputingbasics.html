Types of cloud models

  • Cloud models are basically different business models adapted by cloud providers.
    1. Service models (PaaS, SaaS, IaaS)
      • IaaS - Infrastructure as a Service - full access on server, can install any service.
      • PaaS - Platform as a Service - Azure provides preconfigured server with limited settings.
      • SaaS - Software as a Service - delivers software like Outlook, Gmail.
    2. Deployment models (Public, Private, Hybrid)
      • Public - Users share the same infrastructure for their applications.
      • Private - Services exclusive for a single org, either owned or rented.
      • Hybrid - A combination of both public and private.
