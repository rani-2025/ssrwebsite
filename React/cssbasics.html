<p> vs <span> - Block vs Inline Elements

Key Difference: <p> is a block-level element, <span> is an inline element

Comparison Summary

Feature <p> (Paragraph) <span> (Span)
Display Type Block-level Inline
Line Break Starts on new line Stays on same line
Width Takes full width available Only takes content width
Height/Width Control Can set height and width Cannot set height and width (unless display changed)
Margins Top and bottom margins work Only left and right margins work
Use Case Text paragraphs, content blocks Styling part of text, inline elements

Code Examples

<p> Tag Example (Block-level)

<p>This is a paragraph.</p>
<p>This is another paragraph.</p>
<p>Each paragraph starts on a new line.</p>

<p>This is <span style="color: red;">red text</span> in a paragraph.</p>
<p>You can <span style="font-weight: bold;">bold</span> or

Practical Usage

<!-- Use <p> for paragraphs -->
<p>Lorem ipsum dolor sit amet, consectetur adipiscing elit.</p>

<!-- Use <span> for styling parts of text -->
<p>The price is <span style="color: green; font-weight: bold;">$29.99</span></p>

<!-- Common React/JSX usage -->
<p>Status: <span className="status-active">Active</span></p>

Quick Rules

  • Use <p> for complete paragraphs or text blocks
  • Use <span> to style or target specific parts of text
  • <p> cannot be nested inside another <p>
  • <span> can be nested inside <p>, <div>, or other elements
  • Block elements stack vertically, inline elements flow horizontally
